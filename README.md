# Axios-module
با ماژولار کردن کد هامون میتونی خیلی راحت تر و تمیز تر کد بزنیم و از پیچیدگی کد هامون کم کنیم.

## requirements

### Axios 

```bash
    npm install axios
```

### sweetalert
```bash
    npm install sweetalert
```

### dotenv 
``` bash
    npm install dotenv --save
```
## or 

```bash
    npm install
```
